## 📖 How to Use

### 1. Create New CV
- Click **"Create New CV"** on main screen
- Navigate through tabs to fill your information
- Upload photo and signature (optional)

### 2. Fill Your Information
- **Personal Information** - Basic details, contact, summary
- **Work Experience** - Add multiple jobs with descriptions
- **Education** - Academic background
- **Skills** - Technical, business, soft skills, languages
- **Projects & Certifications** - Portfolio entries
- **Academic Sections** - For research positions
- **Declaration** - Final statement with signature

### 3. Choose Template
- Go to **"Template & Settings"** tab
- Select from 10+ templates
- Choose color scheme
- Pick font family
- Preview in real-time

### 4. Export CV
- Select output formats (HTML, PDF, DOCX)
- Click **"Generate CV"**
- Files saved to `%USERPROFILE%\CVGeneratorPro\Output\`
- Option to open folder after generation

### 5. Save/Load Profiles
- Save your work as `.cvprofile` files
- Load existing profiles to edit
- Perfect for updating CVs over time
